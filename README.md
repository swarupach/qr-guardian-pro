
# 🛡️ QR Guardian Pro

QR Guardian Pro is a modern web application designed to enhance QR code security and user awareness. The platform allows users to scan, analyze, and verify QR codes before accessing their embedded content, helping protect against malicious links, phishing attacks, and unsafe websites.

## 🚀 Live Demo

🔗 Deployment URL: https://qr-guardian-pro.vercel.app/

## 📌 Features

- 📷 QR Code Scanning
- 🔍 URL Analysis & Validation
- 🛡️ Security Risk Detection
- ⚠️ Phishing Link Identification
- 📊 User-Friendly Dashboard
- 📱 Responsive Design
- 🔐 Secure Authentication
- ☁️ Cloud Database Integration with Supabase
- ⚡ Fast Performance with Vite

## ⚙️ Installation

### Clone the Repository

```bash
git clone YOUR_GITHUB_REPOSITORY_URL
cd qr-guardian-pro
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file and add:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_key
```

### Run Locally

```bash
npm run dev
```

The application will start on:

```text
http://localhost:5173
```
